# Unique-Renderer-PyPi-Package

This repository contains the source code and documentation for the unique Renerder Python package, which is available on PyPi. The Unique Renderer is a powerfull tool for rendering any youtube videos and any web page over the Jupyter / colab Notebook.

# **How to Run?**

1. Create a virtual environment

   ```
   conda create -n unique-renderer python=3.8 -y
   ```
2. Activate the virtual environment

   ```
   conda activate unique-renderer
   ```
3. Install the package

   ```
   pip install -r requirements_dev.txt
   ```
